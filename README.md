###What is the app?

The Spark App is designed to give users a small "spark" of inspiration throughout the day. Based on the time entered 
the app suggests a short activity or phrase to encourage productivity 

###How does it work?

The app works by:

1 Taking a time-of-day input from the user
2 Processing the input by assigning it to preset phrases
3 Returning a phrase or suggestion appropriate for that time

For example:

Morning → motivational prompt
Afternoon → productivity suggestion
Evening → reflection or relaxation idea

###How to run the app

1 Download and clone this repository 
2 Open the project in Android Studio
3 Click the Run button
4 Select an emulator or connected Android device
The app will launch and you can start using it
